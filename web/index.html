Paste a Pest or PEG grammar, get a runnable Marser parser in Rust — in your browser, no install.

Output is a starter grammar.rs (or a full Cargo project zip) you can extend with real AST types.

  1. Choose syntax (Pest or PEG), then try an example or open / drag a grammar file onto the left editor
  2. Edit the grammar — Rust on the right updates live
  3. Set entry rule, then copy, download, or share a link

Drag-and-drop a .pest or .peg file onto the left pane. Share link copies a URL with your grammar and entry rule embedded. Download project gives a zip with Cargo.toml, main.rs, and README.

    Supported features & limitations (v1)

    Supported

    • Sequences, choices, repetition (*, +, ?, bounds)
    • Modifiers (@, $, !, _)
    • WHITESPACE / COMMENT implicit skipping
    • Lookahead (&, !), case-insensitive strings
    • ASCII builtins, right recursion via recursive

    Not yet (v1)

    • Stack ops (PUSH, POP, PEEK, …)
    • Left recursion
    • Pest-style Pair trees, spans, or memoization
    • Generated captures default to () — replace them with your AST/output types
